# Swap Fix

مود Fabric بيعمل سواب أوتوماتيكي: طول ما المود مفعّل، تمسك زرار
"سلوت الرجوع" وتضرب left click، والضربة بتروح تلقائيًا على سلوت
الاسبير المحدد في الإعدادات، وبعدين بيرجعك لسلوتك الأصلي فورًا -
من غير ما تحتاج تضغط زرار السلوت التاني يدويًا خالص.

- الإصدار: Minecraft 1.21.11 (Fabric)
- Java 21
- Mod Menu (اختياري - لازم يكون مثبت عندك عشان تظهرلك شاشة الإعدادات)

## طريقة الاستخدام بعد التثبيت

1. من داخل اللعبة: Options → Controls → دور على "Toggle Swap Fix"
   وحدد الزرار اللي عايز تفعّل/تعطّل بيه المود (مفيش زرار افتراضي،
   لازم تحدده بنفسك أول مرة).
2. لو عندك Mod Menu مثبت: من قائمة المودات، دوس على Swap Fix،
   هتلاقي شاشة فيها رقمين: "سلوت الاسبير" و "السلوت اللي ترجعله"
   (افتراضيًا 9 و 8). عدلهم زي ما يناسبك واحفظ.
3. في اللعب: فعّل المود بالزرار اللي حددته، امسك زرار سلوت الرجوع
   واضرب left click عادي - الباقي بيحصل لوحده.

## خطوات الرفع على GitHub (مرة واحدة بس)

1. اعمل ريبو جديد وفاضي على GitHub (من غير README ولا .gitignore).
2. من جوه المجلد ده، افتح Terminal ونفذ:

```bash
git init
git add .
git commit -m "Initial commit"
git branch -M main
git remote add origin https://github.com/USERNAME/REPO_NAME.git
git push -u origin main
```

(غيّر `USERNAME/REPO_NAME` باسم حسابك واسم الريبو).

## هيحصل إيه بعد الـ push؟

GitHub Actions هيشتغل لوحده تلقائيًا (الملف بتاعه موجود في
`.github/workflows/build.yml`) وهيعمل build كامل للمود. تقدر تتابعه من
تبويب **Actions** فوق في صفحة الريبو.

- لو الـ build نجح: هتلاقي ملف الـ `.jar` جاهز تحت تبويب Actions →
  اختار آخر run → **Artifacts** في الآخر.
- لو الـ build فشل: افتح الـ log وابعتلي رسالة الخطأ.

## لو عايز Release جاهز للتحميل مباشرة (بدل ما تدور في Actions)

```bash
git tag v1.0.0
git push origin v1.0.0
```

## تركيب المود بعد التحميل

حط ملف الـ `.jar` جوه مجلد `mods` بتاع Fabric Loader (لازم يكون عندك
Fabric Loader + Fabric API متثبتين للإصدار 1.21.11). لو عايز شاشة
الإعدادات، حمّل كمان Mod Menu (نسخة 1.21.11) وحطه في نفس المجلد.

## ملحوظة عن Mod Menu

استخدمت نسخة 17.0.0-beta.2 بتاعة Mod Menu لأنها أحدث نسخة متوافقة
مع 1.21.11 وقت كتابة الكود ده، وهي لسه beta. لو حصل خطأ compile
يخص Mod Menu تحديدًا في الـ Actions، ابعتلي الرسالة وهظبطها -
أسهل حل بديل هو تثبيت نسخة أحدث لو طلعت.

## اختبار محلي قبل ما تعتمد عليه في اللعب الحقيقي (اختياري)

```bash
./gradlew runClient
```
(محتاج JDK 21 مثبت على جهازك).
